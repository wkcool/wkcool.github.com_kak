Weave：Tinder版的LinkedIn，爱“见人”莫过创投圈儿

日期  2014年11月23日 22:00 由  36kr 编辑 新闻类别  互联网 标签

爱社交莫过创投圈儿，一点都不夸张，跟Weave：Tinder版的LinkedIn，爱“见人”莫过创投圈儿Weave旨在牵线用户初次见面。Weave需要用户使用LinkedIn帐号登录，自然LinkedIn的社交数据也是为用户身份背书的重要依据，一旦成功匹配，双方便可开始聊天，然后就可以约了。Brain告诉我，Weave现在每天成功线下匹配1000多对用户，回访用户有10000人，目前70%的用户在美国，欧洲和亚洲占一小部分。“我们知道对话中提及的时点、地点信息点，如“下午3点”，“某某咖啡馆”。GPS信息也会告诉我们这两个人出现在了同一个地理位置。就现在的行业现状来讲，不管是SF还是北京，最最需要见人的就是创投圈儿了，这也是Weave首切这个市场的原因，第一步它要做的最是匹配到用户想找的双方可以“帮忙的人”，它会根据用户行为历史数据推荐更合适的人，比如某用户倾向于在上面找社交类创业者，系统会尽可能推荐这类人给他，系统也会优先推荐活跃的人，线下见面机率大的人。“只要我们了解你是谁，你想见谁，就有可能发生更多更有趣的匹配，比如你想见扎克伯格，有人知道扎克伯格的姐姐就在附近，你是有可能跟扎克伯格说上话的。Brain说明年他们可能会进入中国市场，“毕竟LinkedIn在中国的体量还比较小。”目前Weave上的中国本土用户还不多，就这情况，我竟然还匹配到一个做社交的创业者。创投圈儿这些需求看到的人还真不是一家，前边有挖掘二度人脉价值的职场社交脉脉，最近还有一个看起来很有搞头的产品叫“”，同样以职场背景信息为基础，用户可在"会会"上直接发布见人贴，“希望聊聊社交产品运营的事，某某咖啡馆，3:00。Brain做Weave前刚卖掉了自己上一家公司，“挣了不少钱，想要找一些好的创业者来投资，于是自己做了Weave这个产品”，为了快速投资好项目而创立了一家好公司，这确实挺酷的，服务于自身需求的产品看起来都比较靠谱。目前该团队共有7人。

.NET 开源了，Visual Studio 开始支持 Android 和 iOS 程序编写并自带 Android 模拟器

日期  2014年11月13日 09:00 由  36kr 编辑 新闻类别  互联网 标签

.NET 开源了，Visual Studio 开始支持 Android 和 iOS 程序编写并自带 Android 模拟器北京时间今天凌晨的 Connect();大会上，多少程序员的假想成为现实。.NET 开源，集成 Clang 和 LLVM 并且自带 Android 模拟器，这意味着 Visual Studio 这个当下最好没有之一的 IDE 正式支持编写 Android 和 iOS 程序。这个开始前多次通过邮件向核心用户预告的会议果然没让人失望：Visual Studio 和 .NET 真正开始走向跨平台化。Nadella 说的“移动为先，云为先”和“找到微软最初的本质”终于连成一线，这家提供开发者工具/平台起家的公司在继用户之后，开始为 Android 和 iOS 阵营的开发者提供服务。.NET 本次的内容涉及 .NET Framework Libraries、.NET Core Framework Libraries 和 RyuJit VM，允许开发者编写运行于 Mac OS X 和 Linux 的 .NET 程序。此外，微软开始像个人开发者提供免费的 Visual Studio Community 2013，相当于原 Visual Studio Professional 免费向个人提供。开源在 MIT 协议下进行，并提供开源保证（patent promis）。此外，发布会上还透露了有关 Visual Studio 2013 Update 4 和下一代 Visual Studio/.NET 预览版的信息。Visual Studio 2015 和 .NET 2015 的预览版包含名为 "Roslyn" 的 .NET 编译器平台（complier platform），支持用 C#、C++ 和 HTML/JavaScript 编写 iOS、Android、Windows 甚至其它平台的应用。此外，新版 Visual Studio 2015 还内置有 Android 模拟器，相比 Google 提供的模拟器，性能自然是云泥之别（VS 的更优）。[本文参考以下来源：winsupersite.com
