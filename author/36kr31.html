Web App和Native App不是生死之争——反思HTML5惨痛的500天和四个谎言

日期  2014年11月25日 11:00 由  36kr 编辑 新闻类别  互联网 标签 column

“体验经济”的盛行，让“用户体验至上”成了互联网公司铁的纪律。通常来说，“用户”的需求会被放在特定“商业”逻辑里，然后选择具体“技术”来实现，既从User Business Tech。也就是说技术作为底层基础、商业逻辑基于技术实现、用户需求被商业逻辑包装后的技术满足。而在HTML5这个事情上，反而技术逻辑成了优先的部分，打着用户需求的幌子满足野心家们的商业需求。谎言一：用户使用一个native app的时候要去App Store搜索，这一过程繁琐不友好。谎言二：native app的更新频繁，用户对更新感到厌烦。另外native app的更新代表着更好的用户体验和更多新的系统功能加入，不断完善用户体验。谎言三：下载和更新native app耗费流量，流量花费影响用户使用做HTML5的人和用HTML5的人没必要用一种商业逻辑去绑架HMLT5技术和用户需求。“不能把对HTML5的商业逻辑的野心凌驾于用户需求和市场大环境之上”。所谓做互联网的人就是把互联网本身当成生意，而用互联网的人是把互联网当成渠道。“用HTML5”的人：拥有其他的业务，HTML5技术和Web App是用来展示自身业务，把微信、手机浏览器等当成众多流量入口之一的用户。互联网是流量的生意，在不同的有流量的入口上布局是聪明的选择。所以在云API的领域HTML5的技术还是有很多可以对接的服务可供选择。

是时候重新认识 IE 了，来看看 ECMA Script 6 成绩 72% 的新 Edge 内核

日期  2014年11月25日 11:00 由  36kr 编辑 新闻类别  互联网 标签

IE6 的辉煌没有人能够忘记，但随着 Webkit 系的崛起，原开发团队被拆散分配至其它产品的 IE 开始被消费者认为是“卡、慢、丑”的象征。终于，微软开始为 IE 打造新的 Edge 引擎。Edge 在最新的 Windows 10 开发者预览版中已经正式对外测试，同时开发者可以通过在此前的桌面和 Windows Phone 版 IE11 里，微软已经做过类似的尝试，让浏览器可以兼容一部分 Webkit API。而 Edge 可以理解为这种兼容背后的完整形态，它不仅是对 Webkit 做出兼容并收，同时也为最新的互联网标准提供积极支持。但根据 MSDN 文档的描述，Edge 自带的兼容模式会长期存在，但理论上只在浏览兼容性名单网站和开启企业模式时才会启用。对于在此之外的网页浏览，Edge 会提供一种逾越以往浏览器 UA 匹配的模式来帮助用户获得更好的兼容性和浏览体验。Edge 提供一种可通用的 UA 字段来适应当下的主流网站，让用户能够免于单独识别旧版 IE 的内容来获取正常适配网页。IE 团队希望依靠 Edge 的这种适应和兼容能力来让用户在未来彻底摆脱“企业模式”这种妥协方案。与此同时，他们还会持续跟进各种互联网标准和其它浏览器，以保证 IE 可以随时跟上市场的变化。根据此前的消息，IE 会推出新的 UI，新增插件模式，并且加快升级进度。结合上述 Edge 信息，新款 IE 给普通用户的使用感受会很接近当下的 Chrome 和 Firefox。除了在用户界面和功能上向主流浏览器靠近，IE 也在尝试给开发者带来类似的体验，新版本已经可以通过在地址栏键入 about:flags 来调整实验性功能设置。同时，新的 Edge 内核新增不少开发者要求已久的特性支持，具体详情可以查看 MSDN 介绍。当然，在 Edge 之前，微软在 IE10 和 IE11 时代也曾给出过类似的言论，仅看 MSDN 一家之言可能并没有很强说服力。

打响了互联网垂直招聘的第一枪之后，国内市场涌现出了一系列聚焦于互联网行业的垂直招聘网站，如创业公司视频+弹幕，周伯通招聘新上线“发现”，增进普通求职者对创业公司的了解过的豆瓣系的创业公司我们希望公司和求职者能够直接进行交流，碰撞出火花。这些内容包括公司的业务、团队的工作氛围、创业公司的故事、创始人的访谈等。这些视频都是由周伯通招聘来完成，面向创业公司免费开放。他们希望这些信息是鲜活、有趣、易分享，无节操的。我们希望用这种方式消除信息不对称”，周伯通招聘告诉“原来创业公司在传统的招聘平台上处于劣势，很难让求职者注意到自己。帮助“水面”下求职者-创业公司进行交互未来，周伯通招聘还会上线“企业相册”、“企业投融资”介绍等功能。“水面”下的企业和求职者彼此之间很难发现对方的存在。周伯通招聘希望把这些“水面”下的创业公司和求职者撮合在一起，除了上线发现频道拉平了大企业与小企业之间的差距，周伯通还用弹幕的机制拉平企业与求职者之间的差距。上述的3个功能上线之后，周伯通的简历日均投递量增加了50%左右。目前平台上已经有9000多家企业。

“艺客”吕淼：网购艺术品是一堂要补的互联网大课

日期  2014年11月24日 19:00 由  36kr 编辑 新闻类别  互联网 标签 创业

“艺客”吕淼：网购艺术品是一堂要补的互联网大课一方面“恩格尔系数”比较高的我们并没有这种消费理念，另一方面也没有特别好艺术品电商直指普通人的需求。我们熟知的艺术品拍卖电商一方面更适合价格不菲之物，一方面真假非我等能辨，还有一个原因是在线坐等好几个小时去拍一件东西简直不符合正常网购的逻辑。不久前我们报道过的”（Noarter)的项目做的就是想解决普通人买艺术品的问题，现在“艺客”已完成天使轮融资，A轮也正在推进当中。也有类似的思路，不过他们希望从艺术家社区切入电商。目前，艺术品交易市场是一个线上化非常低的行业，“”创始人吕淼告诉艺客”想做的就是通过互联网的方式盘活市场存量不久他们还将推出“在线专业导购”服务—据说10美女、10帅哥。这主要是帮住不知道买什么的用户挑品，比如正在装修的用户希望买幅画。如果没有大量资本的推动，这种潮流也不容易起来一堂互联网的课此外，国外类似的产品还有对后者而言则存在尾大难掉的问题，由于有太多利益捆绑，嘉德在线拍卖做了10多年了也没做起来。
