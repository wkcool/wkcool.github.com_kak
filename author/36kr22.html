FIR.im 添加权限管理和数据统计功能，另上线 BugHD 崩溃分析工具

日期  2014年12月16日 18:00 由  36kr 编辑 新闻类别  互联网 标签 创业

FIR.imFIR.im 添加权限管理和数据统计功能，另上线 BugHD 崩溃分析工具他们近期在原有的 FIR.im 应用测试平台中上线了测试用户访客密码、历史版本和数据统计功能。访客密码是对原有用户权限管理体系的增强。用户分级机制是实现内外部人员联系、管理的基础，外部测试用户获取对外可对外发放或需要重点测试单独功能的版本，内部人员则可以获取更细分的迭代版本从而进行测试或参与相关工作。在这些基础的分类之上，还可以进行进一步的小范围测试、大规模公测用户分类。访客密码上线后，开发者只需要将地址和访问密码发放给允许参加测试的用户，用户就能通过地址直接下载测试应用。环节简化还不止这些，依靠同时上线的数据统计功能，开发者已经可后台看到测试应用覆盖的设备数，每个设备的型号、系统版本，以及用户 UUID。用户在申请测试时，开发者只需要添加用户的 FIR.im 账户，系统就会自动关联这个用户的 UUID。FIR.im 在开放 API 之外还上线了 FIR.im CLI （由 Ruby 编写）的命令行工具，开发者可以通过它直接将本地应用上传至 FIR.im 后台并发布、配置全局设置、获取应用信息，或对应用包进行签名。“当我们的工具接入 IDE，就能基于代码为开发者实现很多服务”，王猛说。FIR.im 专门为 BugHD 服务设立单独域名，供开发者使用这个服务。BugHD 包含崩溃分析、用户反馈 SDK 和后台数据查看功能。BugHD SDK 集成入应用后最终占用的大小在 100k 左右，供开发者手机崩溃信息和用户反馈。另外，FIR.im 还有一个双向消息体系，这个体系可以接入不同平台、工具，开发者对其进行定制后可以在常用项目管理工具上实时获取所需消息或发送远程指令。

“生活纪录片”：深度加工手机照片，留住生活中的故事于是，“生活纪录片”（它让用户把挑选出来的照片配以文字，然后一段 “有故事的照片” 集锦就自动生成了：伴着自带的音乐，可以像幻灯那样滑动，也可以类似视频那样播放。用户也可以邀请好友共同编辑一段“生活纪录片”，制作完成后可以分享到微信等社交平台。“生活纪录片”团队告诉我，他们做这件事其实源起“篱笆社区”。那里面有许多纪录例如装修新家、宝宝成长、海外旅游等生活事件的长贴，内容真实有感觉，但是呈现方式杂乱。“生活纪录片”就是这么诞生的。“生活纪录片”的用户就是有写字冲动的文艺青年。“生活纪录片”既不教育用户，也不刻意引导。另外，“生活纪录片”目前还不提供给用户自定义效果。他们相信给用户制定出好的效果，可以保证用户产生内容的质量。至于产品的商业化，“生活纪录片”尝试把作品印成实体书，不过制作成本是一个问题。近期，“生活纪录片”还将上线视频功能。据创始人介绍，产品在无推广情况下，目前有注册用户60万，平均月活跃用户6万，平均每日新增用户1000个，上传作品1000个 。团队目前正进行融资。

微软为Lumia Windows Phone推出含四种新手势的Gestures Beta应用

日期  2014年12月16日 12:00 由  36kr 编辑 新闻类别  互联网 标签 移动 手势

微软为Lumia Windows Phone推出含四种新手势的Gestures Beta应用有研究表明，长期使用触屏手机会对手指软骨造成损害。不过，微软最近的一项研发有可能缓解一下触屏的使用。微软原计划是在今年Windows Phone 8.1的第二次更新时推出这些“3D Touch”手势的，不过现在该公司已经取消更新计划，但是提供了部分手势的该公司正在为Windows Phone开发若干新手势，这些手势可以让用户在不需要接触屏幕的情况下与设备进行交互。这项手势应用名字叫做Gestures Beta ，面向Lumia Windows Phone用户提供。手势主要用于打电话的控制。Gestures Beta以供提供了4种手势，一是来电拿起即接听—安装该应用的用户在接到来电时只需拿起手机放到耳边即可接听电话，不再需要触屏点击（或者滑动）接听按钮；二是放下即静音—用户只需把手机放到桌面正面朝下即可将话筒调至静音；三是翻转即拒接—如果放在桌面的手机有来电但你又不想接听，只需将手机翻过去正面朝下即可拒接；四是放下即免提—把手机放到桌面来电即进入扬声器模式。所有这些手势均可开启或关闭。不过微软指出在Lumia 630、Lumia 635及Lumia 530三种机型上部分手势可能会受限。除了可以让用户悬浮在瓷片上、让该瓷片炸裂为与特定应用相关的若干小磁帖的“MixView”以外，微软仍计划为Windows 10引入部分3D Touch功能。此次推出Gestures Beta背后的动机，也许是微软为了明年推出新版OS征求用户反馈之举。
