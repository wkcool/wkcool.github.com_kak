最后一周的十大新品如期来临，本周侧重在关乎人们衣食住行的智能硬件，其次关注开发者服务和信息获取。临近年关，本以为各路豪杰都储备精力待来年发力，没想到许多豪杰出牌都不按常理的。有的硬是在最后的几天放出一年中最有杀伤力的产品，从以下十大新产品中兴许能看出他们的出招路数。

1.微信 Windows 版

微信 Mac 版发布10个月后，腾讯在平安夜那天终于推出 Windows 版的微信客户端。它的界面风格比较对年轻用户的口味，不过只有最基本的聊天和文件传输功能，查看附近的人、朋友圈、微信支付等还是只能在移动端实现。

登录方面「微信Windows版」和网页版一样，第一次登录需要你打开手机上的微信扫描 PC 端的二维码，以后再登录就更简单一些，只需在 PC 端点击“登录”，然后在移动端确认即可。它支持订阅号和服务号直接搜索，也能够群发消息。默认窗口较小，把窗口移到左右屏幕边缘后会自动占满半屏。打开设置界面后发现“聊天记录备份”几个字赫然在目，不过点击备份弹出的信息是“请插入安卓手机”，令人费解。

「微信 Windows 版」刚刚进入 Alpha 阶段，存在的问题不少。比较多用户提到当新消息出现时没有弹出消息预览窗口，可此前由搜狗代工的微信电脑版是有弹窗的。另外每次退出聊天消息都会清空，如果没有提前备份还是只能回到手机上查询。

国内的设计师常常因为无法在网页中自由使用中文字体而抓狂，由于中文字体本身体积巨大、以IE 6（XP）为首的老旧浏览器（操作系统）又久攻不下，使得在国外已经非常成熟 WebFont 技术也对中文字体的网页嵌入无可奈何。而中文字体压缩器「字蛛」，似乎找到了一个可行的解决办法。

「字蛛」通过分析本地 CSS 与 HTML 文件获取 WebFont 中没有使用的字符，并将这些字符数据从字体中删除以实现压缩，将数 MB 的中文字体压缩成几十 KB ，并生成跨浏览器使用的格式。指定 HTML 文件路径就可以自动化的压缩与转码，无需 JS 与服务端支持，过程中完全无需人工干预，可以方便的集成在前端发布系统中。

日本移动运营商KDDI近日推出了一款搭载 Firefox OS 2.0 系统的智能手机「Fx0」，制造方为LG，设计方为日本著名设计师吉冈德仁（Tokujin Yoshioka），手机 Home 键上印有火狐经典LOGO。也许是秉承 Firefox 系统开放、自由、透明的理念，「Fx0」采用了友商几乎从未考虑过的全透明机身设计，手机内部构造清晰可见，看起来非常酷。

和酷炫的设计相比，「Fx0」的性能配置则略显平庸。它搭载 1.2 GHz 高通骁龙 400 处理器，内置 1.5 GB 内存和 16 GB 存储空间，支持最高 64 GB 存储卡，电池容量 2370 毫安。4.68 寸HD IPS 屏幕，分辨率为 1280×720 ，配有 800 万像素主摄像头和 210 万像素前置摄像头。支持高速 4G LTE 网络、 NFC 和 WebRTC 技术。机身厚度10.5mm，重约148g。

「Fx0」在日本售价5万日元（约人民币2600元），未来如果在其他国家发售价格会更高。作为目前硬件规格最高的 Firefox OS 智能手机，它对于火狐的手机梦无疑意义重大。

4.PrintSnap

人类也许永远在拍照领域欲求不满，这不麻省理工学院的 Michael Ciuffo 花3个月时间折腾出一台名为「PrintSnap」的拍立得相机，它可以用最普通的购物小票现场打印照片。

Michael Ciuffo 做出的「PrintSnap」样品机规格为3.25 x 6.75 x 3英寸，重量1.45磅，看上去有点像便当盒。「PrintSnap」不需要电池、胶卷和内存卡，不需要联网，甚至不需要墨水。可是不要被它看似残缺的身体蒙蔽了，在装上热敏打印纸后它可以用很快的速度打印出分辨率为640×384的黑白照片。照片3 x 1.75英寸大小，在装满50英尺打印纸的情况下最多可打印出150张。

「PrintSnap」的打印成本极低，每张照片仅花费0.003美元，相当于拍立得 600 型相机单张照片成本的千分之一，因此你可以随心所欲的用它来拍照玩耍。但由于该相机尚处于研发阶段，要想买到还得等一段时间。

5.Panda 4

咱们的国宝太受歪果仁欢迎，继接拍两部《功夫熊猫》及担任西班牙杀毒厂商 Panda Security 的特聘大使后熊猫又开始为媒体代言。当然「Panda 4」不是一个普通的媒体，它专注于聚集可以给人灵感启发的高质量内容源，比如 Hacker News 、 Product Hunt 、 Dribble 等，在阅读体验和吸收效率两个方面都有可圈可点的地方。

「Panda 4」针对开发者、设计师、创业者等不同的群体提供了6个不同的内容组合和5种阅读布局。以 El Classico 布局为例，「Panda 4」被分为两个主要的版块，左边的版块较窄，仅显示非视觉方面的内容，确切的说只有内容的标题。右边的版块较宽，仅显示视觉方面的内容，为了提高视觉体验，默认图片的大小为源图片缩略图的两倍。所有内容都按热门程度或更新时间进行排序，当下看不完的内容还可以一键收藏方便以后再看。

「Panda 4」目前仅有“ Web 版”及“ Chrome 扩展”两种产品形态，并且对内容源的选取保持相当的克制，不允许用户提交及创建自己的来源。

6.魅蓝 Note

魅族在2014年的最后几天仍然不消停，这一次他们推出了千元价位的5.5英寸大屏手机「魅蓝 Note」，价位低端配置上却接近主流手机。外观上也不走寻常路，提供白、蓝、黄、绿、粉5种颜色的后盖。

此次「魅蓝 Note」将屏幕比例变回到比较主流的16:9， 1920 x 1080 分辨率（像素密度403 ppi）普遍高于同价位手机，500万/1300 万像素的前后置摄像头。搭载64位的 MT6752 八核处理器，整合 Mali T760 MP2 GPU，2GB LPDDR3内存，16GB 或 32GB ROM，电池容量3140mAh。系统是基于 Android 4.4.4 的 Flyme OS 4.2C，联通或移动4G，单卡槽双卡双待。机身采用了与 iPhone 5C 类似的一体成型设计，宽高厚分别为75.2 x 150.7 x 8.9 mm、重145克。

由于在屏幕分辨率和 ROM 的大小两个方面打破了业界心理防线，可以预见魅族即将迎来友商们的狠狠回敬。不过每一个民族厂商的进步总是好事，普通消费者真没有什么好担心的。

7.Pixsta

称「Pixsta」为 Instagram 最好的第三方客户端之一毫不为过，它试图完善 Instagram 没有做好的 PC 端用户体验。众所周知 Instagram 对图片格式的控制比较严格，在移动端看上去很顺眼的图片在官网上看却偏小。如果你受够了在 PC 上看小图，「Pixsta」的图片墙及大图浏览模式也许能让你好过一些。

「Pixsta」可以安装在 PC 上或安装在 Chrome 上作为独立应用存在。安装完后默认启动小窗口，显示当下最热门的图片，这时浏览模式跟官网差不多。将窗口最大化后进入图片墙模式，最多同时看到10个图片，点击任意图片会看到更大更清晰的版本。切换到分栏模式时则可以看到几组图片和评论同时出现，图片在左边占很大的区域，评论在右边占较小的区域。可以直接搜索用户或图片关键词，「Pixsta」会按关联度给出结果，搜到的图片都可以下载。管理个人中心也是很方便，跟在官网查看自己 Like 过的图片、评论和分享图片等差不多，可惜不支持微博和朋友圈。

还有一个很赞的功能，「Pixsta」可浏览地理位置上离自己最近的用户所发的图片，增强社交上的归属感。「Pixsta」总体用户体验非常不错，遗憾的是不能通过它发图片到 Instagram 上，似乎没有 Instagram 的授权，也没有热门用户探索功能。注：以上任何提到图片的地方已经把视频包括在内。

8.Foodini

磨好土豆泥放入食物胶囊，选择土豆泥饼干食谱，10来分钟后，几块五角星形的土豆泥饼干就“打印”成型，这令人称奇的创作出自于3D食品打印机「Foodini」。「Foodini」由西班牙初创公司 Natural Machines 研发，旨在帮助用户减少枯燥的烹饪流程和节约烹饪时间，并且吃得更健康。

「Foodini」大小类似家用微波炉，内设5个金属胶囊来盛放不同的食材。打印美食前先要把新鲜食材搅拌成泥状后装入胶囊内，6个喷嘴会按照电脑设定的轨迹来回挤压，完成食物的“搭建”。用户可以通过电脑控制面板调整食物的形状和体积等，目前能打印的美食包括面食、汉堡包、鸡块、匹萨、薯片、饼干、巧克力、蛋糕等。不过「Foodini」没有烹煮食物的功能，用户得把打印好的食物加热煮熟才能享用。

「Foodini」预售价1300美元，零售价还没有确定。若按原定规划现在已经开始限量出货，2015年上半年将批量生产。据测试期用户的反馈，他们比较希望「Foodini」改善以下几个地方：适当降价、提高打印速度、能打印更多复杂的食物以及适应非泥状的食材。

作为最常见的出行工具，人们对自行车的依赖程度不可谓不高，恐怕都盼着自行车能更智能更好用些。只是百度的智能自行车尚在研发中，张向东的才刚刚开始，如今国内相对成型的智能单车产品可能就只有BICI（基本概念）刚刚开发出的「启孜」了。「启孜」定位于偏爱城市骑行和郊游的轻奢一族，目前正在预售阶段。

「启孜」使用纤细的碳纤维车架及时下较为盛行的22寸小轮径，体积介于18寸小轮车与28寸普通自行车之间。通体暗黑色系，8公斤车身重量仅相当于普通自行车的一半。自行车内的传感器可以记录的数据包括骑行踪迹、心率，踏频情况等，手机放在启孜手机架上时数据会实时传送过来。蓝牙配对成功后，「启孜」还能通过车把震动与示宽灯来给用户提供导航。车身所需的电力全部来源于前轮发电花鼓，边骑行边发电，电流通过内走线传输。另外，「启孜」设置了遥控解锁，如果被车主以外的人骑走，它会自动发出警报并向车主报告自己的位置。

BICI将「启孜」的预售价格定为基本款3999元、专家款5999元，用料和智能化程度都要比同等价位的传统自行车好上不止一点点。不过老实说很多人在听到一辆自行车的售价比小米手机还贵时心里就在犯嘀咕了，把价格变得更低可能是一条非走不可的道路。

人们对密码普遍抱又爱又恨的态度，爱因为离不开，恨则因为密码的管理与登录实在是各种麻烦。例如密码输错、忘记存储、被盗、输入不方便、收不到二次登录验证码之类的不爽，大家只怕都深有体会。针对这个老大难问题，深圳的一初创团队开发出人脸登录SDK「一登」，为 iOS 和 Android 应用提供人脸登录功能，简化注册和登录流程，让开发者不至于在登录环节就流失大量用户。

依托于机器学习和深度神经网络两个领域的快速前进，人脸识别的准确度和速度在近两年取得了飞跃性的进展，而在基础设施方面摄像头已成为手机标配，使得规模化的人脸登录成为可能。「一登」就是推动人脸登录普及的先行者之一！说不定在不远的将来，用人脸登录就跟扫二维码一样简单畅快。

目前「一登」正在内测中，不提供开放下载，接入 SDK 需在官网填写相关信息提交申请。申请通过后开发者参考技术文档接入SDK，一般10分钟内可完成接入与测试。所有服务完全免费。
