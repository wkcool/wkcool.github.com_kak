出现了多款重量级的新产品，从即时通讯潜在的颠覆者到旅游出行新星、从团队协作微创新到可能成为未来全新交通工具的雏形，从解决人类基础问题的迷你工具到后PC 时代没落贵族颇具新意之举......

话不多说，一起来探个究竟：

1.Wire

「Wire」可能是 2014 下半年最具美感的即时通讯类新产品，由几位 Skype 前资深员工牵头打造，目前已获得 Skype 联合创始人 Janus Friis 的投资。

「Wire」的操作几乎全基于手势，右滑选择联系人，左滑回到个人主页，下滑浮出联系人搜索，开发者希望用户关注内容分享而非应用本身。大部分功能设置都被隐藏在视野之外，没有碍眼的三道杠式“汉堡包”导航图标。

跟大多数产品的第一版不同，刚刚发布的「Wire」规格上已经比较成熟：

A.偏向于私人聊天，支持群组聊天

B.信息可以选择性地保留，不会在 24 小时后过期

C.支持应用内即时通话，通话音质被誉为“业界良心”

D.提供了 iOS、Android 和 OS X 客户端

E.在所有呼叫上都运用了端到端加密技术，全程受到有效保护

F.支持分享文本、图片、GIF、YouTube 视频及 SoundCloud 音乐

用户上传的头像会自动生成暗黑系半透明背景，有 7 种颜色的简单滤镜可选，别人跟你聊天时屏幕背景就变成你的头像。打开应用时会自动出现联系人列表，此时的背景是你上次联络过的人的头像，最常联系的人在屏幕中靠得越前。当有新消息出现时，联系人的左边会出现一个小光点，伴随着表针走动的声音，点击一条消息记录还会跳出具体时间。

「Wire」发布后 NEXT 迅速建立了一个超过100人的应用内交流群“NEXT观光”，根据小伙伴们的反馈，「Wire」让人惊艳的同时存在不少细节问题，如：要搬梯子才能注册成功及发消息；容易误操作；过于简洁以至于不容易上手等。

除此之外操作体验均达到了上乘水准，有趣的是，它提供了类似 Facebook Poke 的小功能 Ping，试图顽皮地唤醒好友，Ping 一下便留下一朵绽放后的花朵。

2.蝉游攻略

同属蝉游记团队的作品，蝉游记侧重在旅行探索、分享游记、享受旅途中的美，「蝉游攻略」侧重在搞定旅途中的各种麻烦事，免得糟蹋好心情。

不同于传统的专家编写指南，「蝉游攻略」内容来自于2014年出行的上千位旅行达人，达人写的每一条贴士都是有报酬的，有助于激励达人写出更高质量的内容。旅行贴士从省钱秘笈到小巷美食，从购物榜单到防坑贴士，从行程规划到腐败心得，遇到什么麻烦都有锦囊应对。图片量很大，既获得帮助，也让用户过去眼瘾。

「蝉游攻略」像是一个 wiki 风格的旅行指南，每一个国家都是一个小百科。自然少不了来自蝉游记中的优秀游记，它们被放在“攻略”部分，“攻略”中还有对目的地的介绍、路线概览、交通等，包含了一般出游需要掌握的信息。

应用集成了13种语言的语音翻译功能，对着手机讲中文，立刻翻译成当地语言。又集成各国实时汇率、当地天气与时差查询及当地实用服务查询。创新的地图画板帮助用户在地图上轻松画出将要出行的路线，不过目前仅 iOS 客户端支持。

不足之处主要在于现在有攻略的目的地不多，有些国家的攻略要分享朋友圈才能打开，旅行地图部分打开时看到满满一堆的地点标志，密集恐惧症患者怎能不抓狂~

阿里巴巴刚刚发布一款针对企业的团队协作工具「钉钉」，试图构建新一代团队沟通方式。没有任务管理、报销、实时协作那些比较重的功能，「钉钉」尝试走比较轻量化的路线。

即时通讯部分，单聊和群聊都可以传输文件，发送文字、语音、图片等。发出去的消息可以追踪到谁没有读，消息未读成员列表一目了然。还有神秘的 DING 传达方式，发出的 DING 消息将会以免费语音／免费短信的通知到对方，让对方即刻做出回应。

依托安全稳定的三大运营商直线网络，「钉钉」提供免费、安全、高质量的多方电话会议。为了解决较大的团队彼此不容易熟悉的问题，「钉钉」集成了企业通讯录功能，找单位同事不再托人咨询联系方式。

IOS、安卓、Mac、PC全平台支持，不过目前仅能企业用户注册，个人用户还得等等！官网

https://www.dingtalk.com/

，感兴趣的企业用户可以尝试下。

4.Hendo Hoverboards

「Hendo Hoverboards」被时代杂志评为2014年25大最佳发明之一，它采用了类似磁悬浮列车的技术，让人在特定情况下短暂摆脱地心引力。

依靠磁场产生的升力，「Hendo Hoverboards」滑板能够悬浮在离地面1英寸的地方，前提是这片地面是由铜、铝或其他除了铁之外的金属所制成。不像电影中的 Hoverboard 轻便和易于携带，它可以说非常笨重，而且运行时有一定的噪音，对地形要求苛刻，并无法很好地控制方向。加上成品10000美元一块，目前只能算个玩物，可是「Hendo Hoverboards」有可能通过以下方式变革世界：

A、摆脱地心引力，加速产生对抗引力的智能设备

B、成为未来某种交通工具的雏形

C、诞生新的体育运动、锻炼方式、艺术形式等

也许不久的将来，每天就能看到哈利波特们在窗外飘来飘去，各种相应的基础设施相应地也会出现。

5.Status

「Status」可以实时显示你最亲密的人他们所处的状态及状态的改变，只要你们的手机都装上了该应用。最亲密的人可以是父母、孩子、男友、女友、基友等，对方的状态包括但不限于：

D.电池电量低

F.骑自行车

G.在会议中

H.没什么动静（off the grid）

由于实时知道对方的状态，就能及时感知到对方是否有危险并及时帮助化解，这对于爱的传递是非常有益的。应用安装后就在后台工作，自动识别你的状态不需要手动设定，对电池续航的影响微乎其微。如果很迫切需要跟对方联络到，当ta脱离忙碌状态时你会立刻收到提醒，必要的时候还可以在应用中发信息及拨打电话。

「Status」提供的另外一重价值是可以减少不必要的信息传送，因为明确了对方的状态，就不需要再次询问对方在干嘛。如果知道对方的电话正在占线，就不会误会对方是不是故意伪装通话中。如果知道对方在开车也不会打电话过去让对方慌乱。

「Status」已获得 Google Ventures 和 SV Angel 120万美元种子投资， Gmail 创建者 Paul Buchheit 也非常看好。但「Status」已知的问题也不算小：国内许多用户表示注册时收不到验证码，另有部分用户声称「Status」对电池续航的影响比官方所说的严重。

6.Typed

著名 GTD 应用 Clear 的开发商 Realmac 刚刚发布了「Typed」： 一款Mac下注重写作体验的轻量化 Markdown 编辑器 。

Markdown 是一种轻量级标记语言，它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的 XHTML （或 HTML ）文档”，让编程接近于书写格式。

看看「Typed」都有什么特点：

A. UI 极简，仅保留最常用功能

B.自动保存

C.支持版本回溯

D. HTML 预览/导出

E.字数统计/限制

F.窗口透明

G.多种快捷键，可自定义

H.内置“冷、暖、夜”三款主题

最大亮点当属 Zen Mode（禅模式）了，提供全屏编辑 + 6种类似白噪音的特效音乐，让你全身心的投入到写作中去、更有效的保持注意力。另外不时蹦出小 Quotes，仿佛每天都有小惊喜待发现。

「Typed」的发布引发了一场小骚动，它具有的美感着实令人印象深刻！不过也有用户表示不满，比如导出 HTML 时几乎没有模版和排版可言、Zen Mode 华而不实、24.99美元（特卖期19.99刀）售价太高等。

7.IBM Watson Analytics

在结束超过2.2万数据分析人员的测试之后， IBM 近日宣布将旗下的「IBM Watson Analytics」开放给个人使用。「Watson Analytics」是一种以自然语言认知为基础的大数据分析系统，结合了人工智能和云计算，可以把它理解为一颗有智慧的超级大脑。

「Watson Analytics」提供自助式可视化分析功能，帮助简化得出结果的流程，通常只需要两个大的步骤：导入数据及提出问题。它甚至能理解用户语无伦次的口语化提问，并且在回答的时候使用大众能理解的语言。特别适合于企业的数据密集型部门将数据准备、预测分析和可视化呈现等环节实现自动化。

「Watson Analytics」的预测分析模块自动挖掘出相关度最高的事实，揭示无法预料的各种模式和关系。它告诉你什么是最有可能的，什么是非常不确定的，能帮助处理一些难以判断准确的问题，比如：

A.哪些关键因素促进了产品销售？

B.哪些交易最有可能达成？

C.什么样的激励计划有助于留住人才？

基于电脑相对人脑在处理速度上的天然优势，「Watson Analytics」让用户不用再遴选大量数据、搜索答案或者检验假设条件时无序摸索，可节约大量时间。《纽约时报》认为「Watson Analytics」可以与PC时代的电子表格和移动计算时代的 iPhone 手机相提并论。

最新发布「Watson Analytics」采用 Freemium 模式，所有人都可以使用该数据分析产品，初级服务免费，进阶服务收费。

8.Dragdis

「Dragdis」是一个浏览器扩展工具，它可以在最短的时间内保存及整理网上的一切内容，包括文字、照片、视频、链接。

保存内容部分：让保存任何内容只需一个连续的步骤：选取、拖曳然后释放。侧边栏让操作更加快速，这个侧边栏并不是始终显示在屏幕上 —— 只有你选择了网页中的对象，开始将其拖至屏幕右侧时它才会出现。目前支持在 Chrome 、 Firefox 、 Safari 3个浏览器上安装扩展，首次通过关联账户登录「Dragdis」时会出现非常简单直观的操作引导。

整理内容部分：在侧边栏建立不同文件夹，对应各种分类，就不用保存完再进行整理啦！每个文件夹都可自定义名称、颜色及图标，并以拖曳的方式调整分类顺序。还支持对内容的社会化分享、添加标签和备注，登录「Dragdis」网站随时查看。

「Dragdis」需要改进的地方：目前搜集到的东东在网站中以小豆腐块呈现，当内容变得很多时就不好回顾和提炼了，最好有自动加TAG等自动化的功能帮用户减少后期处理的时间。

  1. Yandex.Browser alpha

俄罗斯搜索巨头 Yandex 日前推出了旗下基于 Chromium 的新款浏览器的 alpha 版本，它重点借鉴了 Chrome 、 Opera 和 Windows 8 的许多元素，置于底部的标签页让它格外与众不同。

「Yandex.Browser」整体风格极简，地址栏和书签栏被完全隐藏，视野接近全屏模式。点击窗口顶部的网站名称后，才可以看到完整的URL及选择添加到收藏夹。自带扩展已支持截图、广告拦截、印象笔记等，在低速网络中可开启类似 Opera 浏览加速器的“ Turbo 模式”，图片和流媒体都将被压缩体积。

「Yandex.Browser」已支持 Windows 和 Mac OS X 两大平台，暂未发布中文语言包，但做了一些本地化处理，安装后新建标签页会出现百度等8个中文网站。空白标签页置于漂亮的动态背景中，共有12张高清图可选。

10.Outernet Lantern

「Outernet Lantern」是个迷你网络接收器，帮助世界上任何一个角落的人联网，安全、无访问限制、完全免费......

「Outernet Lantern」需要和 Outernet 外联网协作才能上网。「Outernet Lantern」好比汽车里的调频收音机，而由卫星集群组建的太空 Outernet 好比无线电台。「Outernet Lantern」持续接收太空中 Outernet 外联网所传输的无线电波，并将无线电信号转化为数字文件存储在其内置硬盘上。然后打开「Outernet Lantern」中的 Wi-Fi 热点功能，将支持 Wi-Fi 功能的电子设备连接到 「Outernet Lantern」上即可随意浏览。

「Outernet Lantern」的重要特性：

A.极致便携：成品在尺寸和重量上和一般的手电筒差不多

B.太阳能充电：「Outernet Lantern」配有太阳能电池板，可直接在阳光下充电，适应更多恶劣环境

C.流量免费：最初「Outernet Lantern」每天只能接收 2MB 的数据，如果众筹活动中能够募集到 1000 万美元然后实现下一阶段重大目标，全球所有地区的 Lantern 用户都能每天接收100M的数据

此外，由于Outernet 外联网不受地球上天灾人祸的影响，在其他一切联网设备失去作用的情况下「Outernet Lantern」仍可以继续工作，第一时间传播灾难相关信息，为化解灾难带来无限希望。

Outernet 的使命在于确保所有人都能享有最基本的信息平等权利，无论收入高低、身在何处、基础设施是否完善。 Wired 称其「为世界互联网盲区带来信息福利」，它在帮助全人类上网方面跟谷歌、Facebook做的是同样的事情，也会还会做得更好。
