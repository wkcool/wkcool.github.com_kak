Google在自己的官方

发布消息称，到2015年12月12日将停用Google Earth API。此举可能与Chroem和Firefox均停止对NPAPI框架的支持以及Google减少对跨平台支持有关。

2008年推出的

Google Earth API

至今已有6年历史，开发者可利用这一API和JavaScript开发基于浏览器的3D地图应用。其技术基础是名为NPAPI的插件框架。但最近，出于安全的原因，Chrome和Firefox这两大浏览器均先后宣布停止对NPAPI框架的支持。再加上对跨平台支持（尤其是移动设备上）的逐步减少，使得利用Earth API来开发应用变得不再可行。

不过，在2014年12月12日到2015年12月12日这段时间内，那些仍支持NPAPI的浏览器还可以支持Google Earth API。其中包括Windows平台上的Google Chrome 5.0-39.0（32位版本）、IE 7-9，以及兼容视图下的IE 10-11，以及Mac OS X10.6及以后的Google Chrome 5.0-39.0（32位版本）、Safari 3.1+、Firefox 11.0-34。

目前Google尚未发布弃用Earth API后的解决方案。
