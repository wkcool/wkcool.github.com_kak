今天起忘掉 apt-get 吧，Ubuntu 要让服务器系统有媲美 OTA 的更新体验Canonical 透露了有关新版 Ubuntu 的信息，根据描述，这个名为 Unbuntu Core 的版本专用于服务器并对 Docker 使用做出优化，期望给服务器操作系统带来“一键更新”的升级和管理体验。Docker 在主流云计算平台的火爆已经暗示了云资源管理便捷化的趋势和需求。尽管 CoreOS 不断开炮，似乎也无法阻挡这些在云计算领域握有话语权的公司拥抱 Docker。Ubuntu Core 是一个小型服务器操作系统，它使用于现有 Ubuntu 相同的库，同时使用更简便的机制（即容器）供用户安装应用。此外，Ubuntu Core 使用的这种容器机制也兼容 Docker。Cononical 创始人 Mark Shuttleworth 在个人博客里写道：这是 Ubuntu 十年来最大的突破，因为 Ubuntu Core 摒弃了传统的 debs 安装包或 apt-get 指令来安装应用。不同于以往已“包”为基础的服务器或桌面操作系统，全新的部署机制能让应用安装和系统更新的获取和管理都变得更加便捷。与此同时，因为 Ubuntu Core 本质上与现有的 Ubuntu 相同，开发者可以简单调整原有的 debs 安装包来适应新部署机制。在 Liaison Jorge Castro (Canonical Cloud Community) 对 Arstechnica 的描述中提到：与手机更新类似，Ubuntu Core 的系统更新是已完整镜像为单位进行的，系统会在新镜像下载后重启安装更新（或在校验失败时拒绝），而不是传统的下载单独安装包、各自解压，最后逐一安装。这种连续写入的方式也使实际系统更新所用时长得以缩短。Ubuntu 被认为在服务器和企业市场中是 RHEL 有力的竞争者之一，如今 Ubuntu Core 已经在 Microsoft Azure 中提供测试版，同时也可以运行在本地 KVM 虚拟机上。[本文参考以下来源：arstechnica.com第一个对 apt-get 说再见的主流服务器操作系统，会改变市场格局吗？

如果你是一个对空气质量和周围环境很敏感的人，那除了空气净化器，可能你还需要一个可以随身携带的环境监测设备。正好，TZOA能帮到你。TZOA是一款可以监测环境、发送实时空气质量和紫外线指数的可穿戴设备。它通过连接人们和其周围的环境，帮助你提高并改善日常的生活习惯。把TZOA扣在你的衣服上，它就会实时告诉你哪里环境更好，从而帮你规划出行或是骑行路线。这款圆形的小追踪器里配备了传感器，可以检测空气质量、紫外线、湿度等指标。所有数据都会收集到配套的App中，然后进行相应的环境量化分析，为佩戴者提供环境服务。除此之外，TZOA还知道花粉含量什么时候升高、空气质量什么时候变差，或者你是否应该换个地方了。“众包”也是TZOA的特色之一，用户不仅可以快速查看本地空气质量，还可以上传数据创建一个本地污染地图，它能为你标出实时的污染区域和未污染区域，用户可以根据这些信息选择散步或运动的目的地以及路线。当你身边的环境污染等级升高，空气质量下降时，TZOA会发出警告，并建议用户更换场所。目前TZOA仍处于初期阶段，团队还在开发早期原型产品，他们希望可以进一步测试自己的设备以对抗那些高端的空气监测传感器。TZOA将在2015年年中进入生产阶段，如果你感兴趣可以到Kickstarter上去，截至目前，TZOA已筹得了近6万美元。不过大飛要吐槽的是，在经常出现大范围雾霾的区域，TZOA使用起来会很鸡肋，即便它不提醒你，这也都无所谓了。你懂的。

Google主席Eric Schmidt：不用害怕未来的人工智能

日期  2014年12月10日 22:00 由  36kr 编辑 新闻类别  互联网 标签 人工智能 未来

当心人工智能“钢铁侠”Elon Musk说，Google主席Eric Schmidt：不用害怕未来的人工智能自动化会让人类变蠢这些担心是正常的，但是也存在一定程度的误导。Schmidt指出，人类对机器掌控世界的担忧已有几个世纪。“回顾织机的历史（就可以发现），当时存在着绝对的混乱。”而且过去的发展表明，采用的新技术越多，经济就会越繁荣。“有许多迹象表明，计算机出现时，薪水也涨了。他认为，真正的威胁是全球的教育体系并没有给学生传授与这些智能化程度越来越高的机器协作所需的必要技能：用计算机工作的人获得的报酬要比不用计算机的人的多。真正需要担心的，是如何去改善全球的教育体系和激励机制，以便让人为这个新世界做好准备，从而实现个人收入的最大化。Schmidt用一种略带失望的语气说神经网络最终发现了“猫”这个概念，但是人工智能也就仅此而已了。也就是说，Schmidt认为人工操作员的作用仍然与技术一样的重要，AI远没到取代人类的地步。人工智能无疑是Google目前和未来技术的核心，从搜索到无人汽车到机器人，无一不需要仰仗AI。

张小龙将怎样讲微信？重读“微信造人”

日期  2014年12月10日 19:00 由  36kr 编辑 新闻类别  互联网 标签 column

重读“微信造人”曾经和张小龙深度交流过的黑豆，恰巧在一年半以前写过一篇《微信造人》，就是畅想微信如何解放我们的身体，那时5.0版本还没上线呢。微信造“人”虽然朋友圈拍图的设置已经说明，张小龙团队已经在努力降低这些门槛，但门槛还可以更低。因此你可以说，微信正在造“人”，用各种功能替代人繁琐的动作。除了扫描二维码，微信增加了扫描条码、扫描图片、扫描街景和扫描文字功能。扫一扫对此进行了逻辑分类。受技术所限，现在图像搜索技术还不成熟。因此这些“扫”的功能都是帮人们辨别。现在新推出的视频搜索原理也相似，抽取视频的几个要点进行匹配。微信第三块变动是订阅收藏。微信还想简化人的思维过程。订阅收藏至少可以简化这一过程。就像大脑一样，微信既可以收藏图像、也可以储存声音。现在也没人再质疑，微信是信息大入口，搜索功能是被腾讯投资的搜狗来实现的。
